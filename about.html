About Project

This is a interative web demo for Genetic Algorithm in which an algorithm try to create target word given by evolving itself after number of generations. Problems which can take years to solve by simple Bruteforce can be solved in seconds or minutes with Genetic Algorithm. This algorithm is based on biological evolution of life. This demo can be used to teach someone about Genetic Algorithm also it can be helpful to visualize how algorithm bsehaves on tuning or changing some hyperparameters like Mutation Rate and Population Size.
